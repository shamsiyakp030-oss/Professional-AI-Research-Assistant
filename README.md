# 📖 Overview

Professional AI Research Assistant is an intelligent research platform built for researchers, students, educators, and data scientists.

Instead of manually reading hundreds of research papers, users can upload research documents and datasets, then ask natural language questions to instantly retrieve accurate answers.

The system combines modern Natural Language Processing, Large Language Models, Semantic Search, Vector Databases, and Data Analytics into one integrated solution.

## 💬 Document Question Answering

Ask questions like:

- What is the objective of this paper?
- What methodology was used?
- What are the main findings?
- What is the conclusion?
- What are the limitations?

- Sentence Transformers
- FAISS
- FLAN-T5
- Retrieval-Augmented Generation (RAG)

# 🚀 Installation

Clone repository

```bash
git clone https://github.com/shamsiyakp030-oss/Professional-AI-Research-Assistant.git
```

Go inside

```bash
cd Professional-AI-Research-Assistant
```

Create virtual environment

```bash
python -m venv venv
```

Activate

Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```
Streamlit community cloud
```bash
https://professional-ai-research-assistant-uugkszxqkvaooeeaksz56e.streamlit.app/"
```
